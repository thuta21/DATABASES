- Getting start with Docker
     
    1. Install Docker Desktop ( https://www.docker.com/products/docker-desktop/ )
    2. Run this command
     
```
docker run --name test-mongo -dit -p 27017:27017 --rm mongo:4.4.1

docker exec -it test-mongo mongo
```
